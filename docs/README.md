# Development docs

The development record and roadmap for the plugin. For **usage**, see the
[main README](../README.md); for **version history**, see
[`CHANGELOG.md`](../CHANGELOG.md).

Work is tracked as milestones, each with its requirements and a
per-requirement status snapshot:

- [Milestone 1 — Foundations](milestone-1-foundations.md) — correctness and configurability.
- [Milestone 2 — Grafana-native](milestone-2-grafana-native.md) — alerting, annotations, variables, logs.
- [Milestone 3 — Editor, traces, polish](milestone-3-editor-traces-polish.md) — Monaco editor, visual builder, traces, backend resilience, catalog readiness.
